# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0-beta] - 2026-06-09

### Added
- Complete FastAPI backend with Async SQLAlchemy and asyncpg.
- 4 Queue Algorithms: Min-Heap, Timing Wheel, Indexed Priority Queue, Skip List.
- Job dependencies (DAG logic).
- Starvation prevention via linear aging.
- `aiosmtpd` integration for DLQ email alerts.
- React 19 + Vite Frontend with glassmorphism UI.
- Real-time queue length monitoring via SSE.
- Fully typed frontend API client.
- `ARCHITECTURE.md` design document.
